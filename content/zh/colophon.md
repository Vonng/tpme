作为一名工程师，Drew Hoskins 曾帮助 Microsoft、Meta 和 Stripe 设计并构建了多种创新产品与平台。在整个职业生涯中，他始终热衷于赋能开发者，这份热情最初体现在 API 设计上，随后不断扩展。其代表性工作包括：创立并设计了 Meta 的核心 ORM EntSchema；打造了支撑 Stripe 异步服务的中坚系统 Workflow Engine；并参与重构 Oculus VR 的服务技术栈。他还参与了 Stripe Connect、Stripe 的 API、Facebook 的 Graph API、Windows 7 和 Visual C++ 的相关工作。

最近，他加入 Temporal Technologies，担任产品经理，负责改进开发者体验。

他现居旧金山湾区，热衷于竞技桥牌，也喜欢学习技术及其历史。
